# Better Tabs for Unity

**Better Tabs** adds native drag-and-drop tabs to the Unity Editor. Drop folders, assets, GameObjects, or components on any tab bar and Unity opens each one in its own locked native tab.

## Requirements

- Unity 6000.x

## Installation

1. Open **Window > Package Manager**.
2. Click **+** > **Add package from git URL...**
3. Paste `https://github.com/oxitorenk/BetterTabs.git` and click **Add**.

## How to Use

Drag one or more items onto any Unity tab bar

- Folders open in Unity's native two-column Project Browser.
- Assets, prefabs, GameObjects, and components open in Unity's native Inspector.
- Multiple dragged items create one tab each.

Tabs keep Unity's standard editing, preview, context-menu, search, rename, and drag behavior.
