# Better Tabs for Unity

**Better Tabs** adds drag-and-drop tabs to the Unity Editor. Drop a folder, asset, GameObject, or component on any tab bar — it stays pinned as its own tab.

## Installation

1. Open **Window > Package Manager**.
2. Click **+** > **Add package from git URL...**
3. Paste `https://github.com/oxitorenk/BetterTabs` and click **Add**.

## How to Use
Simply use **Drag and Drop** to create new tabs. Click and hold an item, drag it over to any native Unity tab bar (e.g., next to the "Scene", "Game", or "Inspector" text headers), and drop it. A new Better Tab will instantly dock itself there.
